Pay4Day
=======

With Pay4Day it is possible that the players pay money for day, night, sun, rain and storm.

Pay4Day on Bukkit: https://dev.bukkit.org/projects/pay4day

# Commands

* /pay4day
* /pay4night
* /pay4sun
* /pay4rain
* /pay4storm

# Permissions

* pay4day.use.classic
* pay4day.use.day
* pay4day.use.night
* pay4day.use.sun
* pay4day.use.rain
* pay4day.use.storm

# Requirements

* [Vault](https://dev.bukkit.org/projects/vault) >= 1.6
